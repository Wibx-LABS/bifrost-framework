# Bifrost Quickstart

Welcome to Bifrost. This guide helps you get started quickly.

## Token Economy (Optional)

New features compress output by default. Three ways to optimize further:

1. **Enable model routing** in `.bifrost/AUTONOMY.md`
2. **Install Serena plugin** (optional, improves codebase search ~5%)
3. **Install Superpowers plugin** (optional, adds `/brainstorming` command)

Without plugins: framework works at full capacity, just uses built-in logic.
With plugins: better UX, marginal token savings.

See `docs/TOKEN_ECONOMY.md` for details.
